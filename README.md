# pips-guide-dog-journey
A comprehensive, interactive educational game designed for English Language Arts (ELA) and ESL students. Based on the animated short film "Pip," this web-based lesson helps students practice reading, writing, listening, and speaking skills through a series of engaging mini-games.

The application includes 11 interactive learning modules:

Story Quiz: Test reading comprehension with multiple-choice questions about the plot.

Sentence Scramble: Reorder words to form correct sentences from the story.

Fill in Blanks: A cloze activity where students complete the text with missing words.

Word Scramble: Unscramble letters to spell key vocabulary words.

Grammar: "Adjective & Noun Hunter" - A two-step game to identify describing words and the nouns they modify.

Dictation: Listen to audio sentences and type exactly what you hear (strict spelling/punctuation).

Story Reading: A dedicated section to read the full narrative of Pip's journey.

Speaking: "Hero Talk" - Prompts to encourage verbal storytelling with guiding questions for self-assessment.

Writing: "Pip's Diary" - A creative writing prompt with a vocabulary bank.

Memory Match: A classic card matching game with story-themed icons.

Leaderboard: A local scoreboard to track top quiz scores and memory moves.
